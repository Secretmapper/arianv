Create Pong with phaser.js

May 6, 2015 · 2 minute read · Comments

In this tutorial, we are going to create a complete Pong game using Phaser.js

Setting up the project

The first thing we need to do is to initialize our Phaser game instance. This is done by adding the line:

var game = new Phaser.Game(400, 300, Phaser.AUTO);

The first parameter is the width of your game, while the second one is the height. In this case 400 x 300 pixels. The third one decides which rendering context to use.

We’ve created the game instance! If you run the code above, you should see a black box. Great, but we need to do more!

The next thing we need is to do is to add a state. We add states with the method game.state.add, the first parameter being ‘play’, which the name of our state, while the second parameter is an object that contains three important functions: preload, create, update. We will get into more detail about these functions later on.

var playState = {};
game.state.add('play', playState);

We then need to call the game.state.start method to actually start our state, passing the name parameter we used a while ago.

game.state.start('play');

Our Phaser Game Instance now has a state, and it starts it, but it doesn’t really do anything yet. Let’s change the Background Color of the game first.

var playState = {
  preload: function() {
    game.stage.backgroundColor = '#71c5cf';
  }
}

var game = new Phaser.Game(400, 300, Phaser.AUTO); var playState = { preload: function() { game.load.baseURL = '/media/images/pong/'; game.stage.backgroundColor = '#71c5cf'; game.load.image('paddle', 'paddle.png'); }, create: function() { game.add.sprite(0, 0, 'paddle'); }, update: function() { }, restartGame: function() { game.state.start('play'); } }; game.state.add('play', playState) game.state.start('play');
